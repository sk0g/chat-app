Routes

    Groups

  • GET /
    • Returns a list of all the existing Groups
  • POST /
    • Adds a group with the provided name
  • GET /:groupName
    • Returns the specified group
  • DELETE /:groupName
    • Deletes the specified group

    Users

  • POST /
    • Not fully functional, creates a new user
  • GET /
    • Not fully functional, gets a list of all users
  • PATCH /:userName
    • Not fully functional, edits an existing user
  • DELETE /:userName
    • Not fully functional, deletes an existing user
  • GET /:userName
    • Not fully functional, retrieves a user
